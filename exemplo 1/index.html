Busca Usuários do GIT

Atividade

Utilizando o recurso de AJAX e a API do GIT (https://api.github.com/users/{nome do usuário}), implemente a funcionalidade de busca geral de usuários pelo nome, buscando e completando o resultado com os valores do usuário referentes a: foto, Nome, Bio, Blog e Location.
